Instruções

Neste protótipo estão disponibilizadas as seguintes funções:

  • 1. Consulta de estados da casa:
  • Menu Principal -> Estados da Casa
  • 2. Controlos de iluninação:
  • Menu Principal -> Funcionalidades -> Luzes
  • 3. Ajustar os horários de rega automática:
  • Menu Principal -> Funcionalidades -> Regador
  • 4. Definir as temperaturas dos ares condicionados em vários locais da casa:
  • Menu Principal -> Funcionalidades -> Ar Condicionado
  • 5. Ativar a alerta de intruso:
  • Menu Principal -> Notificações, clique em "Emergência", vai aparecer um botão ao lado do "Intruso", clique-o. A desativação é feita da mesma maneira.
  • 6. Consulta e gestão de lista de notificações de intruso:
  • Menu Principal -> Notificações -> Emergência -> Intruso
  • 7. Restabelecer as pré-configurações do sistema.
  • Menu Principal (clique no segundo ponto abaixo) -> Configurações -> Restabelecer Sistema
